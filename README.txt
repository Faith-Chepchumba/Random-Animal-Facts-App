# Random Animal Facts App

## Overview

A Python program that provides random fun facts about animals. Users can generate a new fact by pressing Enter.

## Features

- Press Enter to receive a random animal fact.
- Contains a variety of fun and quirky facts about animals.

## Technologies Used

- Python

## How to Run

1. Run `random_animal_facts.py` using Python: `python random_animal_facts.py`.

2. Press Enter to get a random animal fact.

3. Type 'quit' to exit the program.
